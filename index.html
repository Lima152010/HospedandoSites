Passo para Hospedar um projeto no GitHub Pages

  1. Crie um repositório no GitHub com o mesmo nome do seu projeto
  2. No terminal, inicialize o repositório local com git init
  3. Altere o nome da branch principal para main
  4. Adicione os arquivos com git add . e faça o commit git commit -m 'Fist commit"
  5. Conectte ao repositório remoto com:
    git remote add origem https://github.com/seu_usuario/nome_do_repositorio.git
  6. Envie os arquivos com git push -u origin main .
  7. No GitHub, vá em Settings > Pages
  8. Em "Branch", selecione main e a pasta / (rost). Clique em Save
  9. Após alguns segundos, seu site estará disponivel no link exibido em "Pages"
